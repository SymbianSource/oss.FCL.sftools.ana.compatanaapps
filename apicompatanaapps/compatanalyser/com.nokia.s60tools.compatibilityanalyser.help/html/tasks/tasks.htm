Tasks

The following tasks are available for this tool:

  • Selecting the analysis type, baseline profile and current SDK
  • Creating or editing the baseline profile
  • Selecting header files for analysis
  • Selecting library files for analysis
  • Configuring report file options
  • Configuring the knownissues.xml file
  • Saving the analysis into a configuration file
  • Viewing the reports
  • Filtering existing reports
  • Setting Compatibility Analyser preferences
