Prerequisites for use

Before you can use Compatibility Analyser, check that the following items are installed on your workstation:

  • Python 2.4–2.x
  • Minimum of one SDK
  • GCCE or RVCT toolchain for ordinal (library) analysis.

    Note: The GCCE toolchain is installed when you install a Symbian SDK and is typically found in: C:\Program Files\CSL Arm Toolchain\bin. If you download the latest version of GCCE from the Internet, the installation directory is: c:\Program Files\Sourcery G++ Lite\bin.

GCCE and RVCT toolchain

The purpose of toolchains is to help in the creation of .lib and .dso files, to help in the analysis of existing .lib and .dso files, and to make content of the library files readable.

GCCE/RVCT (RealView Compilation Tools)

  • Required mainly in the creation of library (.dso/.lib) files.
  • Also required for analysing libraries (.dso/.lib), which are built using these toolchains.
