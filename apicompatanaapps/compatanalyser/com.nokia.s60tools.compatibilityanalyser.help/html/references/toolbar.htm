Toolbar

The toolbar in Compatibility Analyser contains the following items:

Button Name When to use?
Compatibility Analyser Wizard Opens the Compatibility Analyser wizard. Use it to start the analysis of headers/libraries.
Report Filter Wizard Opens the Report Filter wizard. Use it to start filtering existing compatibility reports.
Related tasks
  • Selecting the analysis type, baseline profile and current SDK
  • Filtering existing reports
