RELEASE NOTES - BCFILTER v1.3.0
RELEASED 17th February 2009 

===============================================================================

Product Description:
====================
BCFilter is a command line tool that can be used to check XML report files for
known issues in S60 platform.

Main Features:
==============
- Support XML report files generated by both HeaderAnalyser and OrdinalChecker
- Insert status information and comments from an XML file containing known
  issues

===============================================================================

What's New in v1.3.0
====================
- Feature: Information to end-user if the issue was found previously but the 
  header has some other unrelated changes
- Fix: Using DOM parser, the checksum comparison was incorrect

===============================================================================

Version History:
================

Version 1.2.0 - 24th October 2008
--------------------------------
- Feature: SAX parser is now being used for large issues files for better
  performance
- Feature: BCFilter can be now compiled and used in Linux environment
- Feature: Added support for the new OrdinalChecker report format
- Fix: BCFilter reserved full CPU timer which made the other system
  unresponsive

Version 1.1.6 - 12th August 2008
--------------------------------
- Change: Shortname tag creation is now done in HeaderAnalyser instead of
  BCFilter
  
Version 1.1.5 - 9th July 2008
-----------------------------
- Change: If no issues are found from report file, a warning is displayed
- Fix: Newline characters were not correct in console output

Version 1.1.4 - 29th May 2008
-----------------------------
- Feature: If the known issues file contains a DTD, it validates the known
  issues file against it and reports any problems
- Feature: Introduced an optional parameter for defining path for the output
  file which allows the orginal report file to remain untouched
- Feature: Progress information displayed in console  
- Fix: Comparing file names was not case insensitive

Version 1.1.3 - 15th May 2008
-----------------------------
- Feature: Platform version tags are appended to the report file

Version 1.1.2 - 11th April 2007
-------------------------------
- Fix: No error was reported if given input files did not exist
